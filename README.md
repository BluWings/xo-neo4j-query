xo-neo4j-query
==============

Additional Query Support for [eXtendend Objects (XO)](https://github.com/buschmais/extended-objects)-Neo4j Datastore

Query Languages
---------------

* Gremlin
* Lucene


License
-------

Build
-----

Start the Maven build on command line

    mvn clean package
